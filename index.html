    Automatic wallpaper changer for Windows

    Project maintained by rhssiiy Hosted on GitHub Pages — Theme by mattgraham

    Любите сюрпризы и неожиданности? WallpaperChanger - это автоматический обновлятор обоев для Windows. Обои обновляются каждый час случайным образом. Источником обоев служит замечательный сайт http://wallbase.cc.

    Requirements

    • Windows XP/Vista/7/8 (32 or 64 bit)
    • Microsoft .NET Framework 4 or above
